# Dotfiles

### Packages Include
- hyprland
- waybar
- neovim
- pywal
- kitty

### How To Install
1. Clone Repository
   ```bash
   git clone https://github.com/ESiegman/dotfiles
   ```
2. Navigate to Repository
   ```bash
   cd dotfiles
   ```
3. Run Install Script
   ```bash
   ./install.sh
   ```
